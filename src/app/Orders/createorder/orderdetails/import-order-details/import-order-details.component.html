Import BBS from Excel (从 Excel 导入钢筋加工表)

Only import active sheet with below format or NSH OES Excel File (仅导入所选择的表格并符合以下的格式, 或者大众钢铁的 OES Excel 文件):

ELEMENT MARKING
构件名称 		BAR MARK
铁代号 		Bar Type
型号		 Size
直径		 Member Qty
构件数		 Each Qty
单件数		 Total Qty
总数		 Shape Code
图形码		 A B C ...
BEAM P1 T 32 2 5 10 038 500 700 600 ..

Header is optional, but without header, the parameters shall appear in sequance. For NSH OES Excel file, if there are more than one BBS in the file, The BBS No to be imported in the file have to be consistent with the BBS No in the system. (表格标题是选择性的. 如果没有表格标题, 参数值必须按顺序依次给出. 对导入大众钢铁的OES Excel文件而言, 如果文件中包含多个BBS, 那么文件中的BBS号码必须与此系统中的BBS号码相同.)
