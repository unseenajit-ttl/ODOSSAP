Clone Order (克隆订单)

The function will clone new orders from current order. After clone, you can find them from Drafted Order of destination project. Please select destination Project, Block, Storey Range and Part of orders you would like the system to clone. 此程序将从现有订单拷克隆新的订单. 新订单克隆后, 您可从所选工程项目的保存订单选项(Drafted Orders)找到它们. 请输入您要克隆订单的工程项目,大牌,楼层范围及分部.
